Case Study
Banking

Problem

A banking team had access to rich data but lacked the internal expertise to build and ship data science solutions independently.

Solution

A hands-on training program built around real business datasets, covering Python, SQL, and Azure from fundamentals through to production delivery.

Impact

9+ Analysts
Upskilled in Python, SQL & Azure

Stack

Python · Pandas · Scikit-Learn · NumPy · Azure ML Studio · Matplotlib

Context

The client is a consultancy that builds internal data science capability at mid-to-large organisations. This engagement was a structured training program for a team of 9 analysts at a banking institution, professionals with strong domain expertise but no prior exposure to programming or machine learning.

The solution

A curriculum was built from scratch spanning Python fundamentals through Azure ML Studio deployment: environment setup, data libraries, statistical foundations, and the full Scikit-Learn landscape across regression, classification, and unsupervised learning. All materials (code, presentations, assignments, and projects) were purpose-built for this engagement, grounded in real datasets from the bank's domain. Azure ML Studio was integrated as the production environment, with workspace setup, training pipelines, and deployed inference endpoints covered as live exercises.

Outcome

All 9 analysts completed the program able to independently scope, build, and deploy data science solutions within the bank's Azure environment. The program produced a reusable curriculum subsequently used in other client engagements.
