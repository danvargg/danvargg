Case Study
HealthTech

Problem

A wellness platform was failing to surface relevant content, resulting in low engagement and high churn.

Solution

A recommendation engine trained on behavioral signals that matches users to content based on mood, history, and context in real time.

Impact

97% Accuracy
Driving high user retention

Stack

Python · Scikit-Learn · AWS Lambda · REST API

Context

The client is an audio streaming platform for meditation and wellness content. As the catalogue expanded, a generic "most popular" approach to content surfacing was no longer sufficient - users were cycling off without finding content that resonated with them.

The solution

Two models handle two fundamentally different user segments. For new users, a collaborative filtering model using SVD surfaces content based on population-level similarity patterns. For returning users, a Random Forest Regressor trained on content attributes and individual interaction history, validated against KNN before selection for accuracy and interpretability. Both models are deployed as independent REST APIs on AWS Lambda, each returning a ranked array of content IDs ordered by predicted rating.

Outcome

The system delivers near real-time personalized recommendations for every user session, replacing a static discovery interface with a ranked feed tailored to each individual. The recommendation engine achieved 97% accuracy on held-out evaluation sets.
